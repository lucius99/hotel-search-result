# Hotel Search Result

### Clone and Run
- Clone this repo to your local machine using `https://github.com/lucius99/hotel-search-result.git`
- Change directory to the cloned repo
```shell
$ cd hotel-search-result
```
- Install npm packages
```shell
$ npm install
```

- Run App
```shell
$ npm start
```
